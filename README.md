# Resume Generator
More accurately a cut and paste shortcut.
Used to choose sections of a "master" template to copy into smaller useable version.
Designed for the "[Resume in Latex][https://www.overleaf.com/latex/templates/cs-slash-it-slash-swe-resume-template/ncxgzcgknkmf]"
